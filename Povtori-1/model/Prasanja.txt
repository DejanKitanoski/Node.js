JSON Web Token

клиентот го чува токенот 

при секое барање до заштитена рута, клиентот го праќа токенот

9.Што е MongoDB? Објаснете накратко за каков софтвер станува збор?
MongoDB е NoSQL база на податоци која чува податоци во форма на JSON документи


10.Што е разликата помеѓу MongoDB и Mongoose?
MongoDB е самата база на податоци
Mongoose е Object Data Modeling библиотека за Node.js која овозможува моделирање и работа со MongoDB
